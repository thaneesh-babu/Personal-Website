DayMaker

A web application that automatically creates Google Calendar events by extracting relevant titles and corresponding dates from uploaded documents using Google Cloud's DocumentAI and NLP API.

This project won the "Best Use of Google Cloud API" challenge at HackTX 2021.

Dilemma Discord Bot

A Discord bot, verified by Top.gg, the largest space of discord bots, that solves typical dilemmas using RNG in multiple servers, totaling 100+ users, with a 5-star user feedback. Additional features include checking compatibility of two names and suggesting random names using APIs.

LibTrack

A Python program that manages and tracks library books in circulation, with a SQL database used to store and retrieve member credentials. The project was briefly tested by the school library department with over 250 books for a period of 45 days.
